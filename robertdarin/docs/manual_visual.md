# Manual visual — Plataforma Fintech Premium

## 1. Concepto visual general
- Fondo oscuro degradado estilo Windows 11.
- Tarjetas translúcidas con bordes redondeados y sombras suaves.
- Botones azules con íconos claros y texto legible.
- Tipografía moderna (Roboto), tamaños grandes en títulos.
- Botón flotante de chat siempre visible en pantallas clave.

## 2. Dashboard
- Sección superior con KPIs: clientes, préstamos, tandas.
- Tarjeta central con gráfica de actividad.
- Botones de acceso rápido: Clientes, Préstamos, Tandas.
- Todo dentro de `PremiumCard` y `PremiumScaffold`.

## 3. Módulo de Clientes
- Botón “Nuevo Cliente” arriba.
- Tarjeta con listado de clientes.
- En detalle de cliente: datos básicos + botones “Editar” y “Ver Préstamos”.

## 4. Módulo de Préstamos
- Botón “Nuevo Préstamo”.
- Tarjeta con listado de préstamos.
- En detalle: monto, plazo, estado + botones “Registrar Pago” y “Ver Expediente”.

## 5. Módulo de Tandas y Aportaciones
- Pantalla de Tanda: nombre, monto, participantes.
- Botón “Registrar Aportación”.
- Historial de aportaciones en tarjeta.

## 6. Módulo de Documentos y Comprobantes
- Botón “Subir Documento” / “Subir Comprobante”.
- Tarjeta con listado.
- Detalle: nombre, fecha, tipo, monto, método.
- Botones “Descargar” y “Eliminar”.

## 7. Módulo de Auditoría
- Tarjeta con registros de auditoría.
- Botones de filtro: por usuario, por entidad.
- Botón “Descargar Auditoría”.

## 8. Módulo de Usuarios, Roles y Permisos
- Usuarios: listado + botón “Crear Usuario”.
- Detalle de usuario: nombre, correo, rol, botones “Editar” y “Asignar Rol”.
- Roles y permisos: listado de roles + botón “Crear Rol” y “Asignar Permisos”.
- Detalle de rol: nombre + lista de permisos.

## 9. Módulo de Inventario, Facturación y Entregas
- Inventario: productos, stock, botón “Agregar Producto”.
- Facturación: facturas emitidas, botón “Crear Factura”.
- Entregas: entregas programadas, botón “Nueva Entrega”.

## 10. Navegación
- Menú lateral (Drawer) con acceso a:
  - Dashboard
  - Clientes
  - Préstamos
  - Chat
  - Panel Admin
  - Usuarios
  - Sucursales
  - Inventario
  - Facturación
  - Entregas
  - Reportes
- Bottom bar con:
  - Inicio
  - Clientes
  - Préstamos
  - Chat

## 11. Botón flotante de chat
- Siempre visible en:
  - Dashboard
  - Préstamos
  - Pagos
  - Documentos
  - Comprobantes
  - Auditoría
  - Inventario
  - Facturación
  - Entregas
- Ícono de burbuja de chat, color azul, sombra suave.
